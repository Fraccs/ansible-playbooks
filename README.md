## Get Started

> On a fresh, minimal Ubuntu 22.04 LTS installation:

```sh
sudo apt update

sudo apt upgrade

sudo apt install git ansible

git clone https://github.com/fraccs/ansible-playbooks.git

cd ansible-playbooks/playbooks && ansible-playbook -K <laptop | desktop>
```
